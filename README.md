# Asp.Net-Core-Stock-Management-System
Asp.Net Core ile geliştirilen bir stok yönetim uygulamasıdır.


# Features

- Asp.Net Core Mvc
- EF / Entity Framework
- Code First
- C#
- Beautiful Bootstrap
- Web API + JQuery
- SendGrid
- Upload Profile Picture
- Edit Profile
- Change Password
- Users, Roles and Memberships
- 25+ CRUD Forms
